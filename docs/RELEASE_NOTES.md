## v1.1.0 (2026-05-07)

### 새 기능

#### 조건 수정하기 / 처음부터 다시하기
결과 페이지에서 두 가지 방식으로 다시 시도할 수 있습니다.

- **← 조건 수정하기**: Step 2(졸업요건 입력) 화면으로 돌아가며 이전에 입력한 값이 그대로 유지됩니다. 항목을 부분 수정해 결과를 재확인하거나, 학우 것을 대신 입력해볼 때 편리합니다.
- **처음부터 다시하기**: 대학원·학과·입학년도 선택부터 초기화합니다.

#### 필수과목 이수 확인
대학원별로 졸업에 필수적인 과목 이수 여부를 Step 2에서 입력받고 결과 화면에 항목별로 표시합니다.

| 대학원 | 필수과목 |
|--------|----------|
| 선교대학원 | 권역별연구 (외국인 학생은 선택과목으로 대체 가능) |
| 상담대학원 | 상담실습 I, 상담실습 II |

#### 이수학점 기준 정정
2026 대학원 가이드북 기준으로 아래 학점을 수정했습니다.

| 대학원 | 학과 | 수정 전 | 수정 후 |
|--------|------|---------|---------|
| 교회음악대학원 | 교회음악학 | 36학점 | 38학점 |
| 통일개발대학원 | 기독교통일개발학 | 36학점 | 30학점 |
| 상담대학원 | 성경적상담학(논문) | 36학점 | 24학점 |
| 상담대학원 | 성경적상담학(논문대체) | 36학점 | 30학점 |

> 상담대학원은 논문 이수방식 선택에 따라 요구 학점이 달라집니다. 논문 선택 시 24학점, 논문대체 선택 시 30학점이 적용됩니다.

### 버그 수정
- 특정 대학원에서 논문 지도 횟수 필드가 유효성 검사에서 누락되던 문제 수정

### 졸업요건 점검 항목

**공통**
- 등록학기 수 (최소학기 충족, 재학연한 이내)
- 이수학점 (입학년도별 기준 적용)
- 평균평점 3.0(B0) 이상
- 자격시험: 영어 / 전공 (과목 수는 대학원별 상이) / 교직 / 전공언어 / 제2외국어
- 논문 이수방식 선택 (논문 / 논문대체 / 졸업연주+소논문)
- 학술대회 발표 1회 또는 참석 2회 이상
- 연구윤리교육(알파캠퍼스) 이수
- 연구윤리와 논문작성법 과목 이수

**논문 선택 시 추가**
- 논문 지도 횟수 (최소 5회·박사 10회)
- 논문 공개발표 완료
- 논문 유사도 검사 통과 (10% 미만)

**박사과정 추가**
- 논문 제안서(프로포절) 심사 통과
- 학술지 논문 게재 (한국연구재단 등재(후보)지 1편 이상)

**대학원별 필수과목** (해당 대학원만 표시)
- 선교대학원: 권역별연구
- 상담대학원: 상담실습 I·II
